Quick Usage

You can use MKView in two ways:

  1. Enter a GitHub markdown URL in the textbox above
  2. Append any GitHub markdown URL to: https://mkview.tech/

Example: https://mkview.tech/https://github.com/username/repo/blob/main/README.md
